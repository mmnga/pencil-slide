# カラーパレットのpencil用penを作成します。

1. 指定されたファイルにカラーパレットを作成してください。

2. カラーパレットの新しい領域を確保してください。そこに新しいカラーパレットを形成します。

3. まずトーンの相性が良い2色を選んでそれぞれグラデーションを作成してください。
    - グラデーションは8パターン用意してください。

4. 追加でグレースケールのグラデーションを作成してください。
    - グラデーションは8パターン用意してください。

5. アクセントカラーを5色選んでください。

6. カラーパレットのレイアウトを調整してください。

7. 作成したカラーパレットを確認して調整してください。色合いを調整して納得が行くものが出るまで調整してください。
