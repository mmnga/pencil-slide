# 指定したテーマについてスライドを作成したい

1. タスク用にtasks/slide_{number}_{今回のタスク名[A-z0-9-_]}というディレクトリを作成してください。

2. theme.md, slide.pen, slide.md, theory-of-slide-design.mdの空ファイルを作成してください。
  - 全てのファイルが作成されているか確認してください。

3. テーマに触れる前に、パワポ、スライド資料の構成手法などでセオリーを調べて根本的な本質や勘所を詳細とともにまとめ、theory-of-slide-design.mdを作成してください。
  - フォントやそれぞれのオブジェクトのバランスや表現についても詳細にまとめましょう。
  - 文字の大きさや強調、認知を意識してください。
  - ここはテーマは関連しません。

4. ではテーマについて詳細を調べて深く考えてtheme.mdを作成してください。
    - テーマの主なポイントを詳しく説明してください。
    - さまざまな可能性、網羅的、深い考察を導き出すために尽力してください。

5. theme.mdを元に10枚程度のスライド用に内容をまとめ、slide.mdを作成してください。
    - スライドは見る人が理解しやすく、ストーリー性などを考慮した構成を考慮して作成してください。
    - 公開資料として素晴らしいパワポ、充実したスライドとなるように考慮してください。
    - レイアウトの構図は見る人がすぐに理解できるように考慮してください。

6. MCPのPencilを用いてslide.penに対して、theory-of-slide-design.mdのデザインを参考に共通コンポーネントを作成してください。
  - BaseComponentsという名前の領域にしてください。領域は十分に広めに確保、さまざまなコンポーネントを考慮してください。
  - まずテーマに合うColorPaletteを作成してください。
  - スライド背景はライトな背景にして、テーマに合うようにデザインしてください。
  - 共通コンポーネントとは、タイトルのサンプル、サブタイトルやカード、表など、スライドの見た目を統一するために定義する部品サンプルです。

7. slide.mdを元にslide.penに対し、theory-of-slide-design.mdのデザインを参考に統一感のあるデザインで全てのスライドを作成してください。
  - その場合、BaseComponentsを参考に、スライドの見た目を統一するようにデザインしてください。
  - 画像やイラストも考慮してテーマに沿ってデザインしてください。

8. レイアウト崩れをチェック、色のバランスをチェック、修正箇所があれば修正してください。

9. 全ての画面を作成したか、修正箇所がないかをチェックしてください。
  - 足りない画面がある場合は、続けて作成してください。

10. レイアウト崩れをチェック、色のバランスをチェック、修正箇所があれば修正してください。
